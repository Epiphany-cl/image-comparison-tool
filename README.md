# 图片对比工具

[在线预览](https://epiphany-cl.github.io/image-comparison-tool/)

[备用链接](https://image-compare.chenlong716.dpdns.org/)

一款在线图片对比工具，支持同步缩放、平移和多种上传方式。

适用于设计师、摄影师等需要精确比较图片细节的用户。

## 功能特性

- 双图并排实时对比
- 同步缩放和平移操作
- 支持拖拽上传文件
- 深色模式支持
- 触控板支持（捏合缩放、双指滑动平移）

## 快速开始

### 环境要求

- Node.js 18.x+

### 安装启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 查看应用

### 构建部署

```bash
# 构建生产版本
npm run build

# 运行生产版本
npm start
```

静态文件将生成在 `out` 目录
